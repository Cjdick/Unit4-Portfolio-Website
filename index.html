Portfolio Projects

Below you will find examples of basic projects I have created. The main purpose of these samples is to showcase a few completed projects. They are not meant to comprehensively demonstrate all of my capabilities.

Small Business Website

A static landing page for a small business. This project was wireframed by myself and showcases CSS/HTML.

Shopping List

Writing a "To-do list"-type app is a common step for students learning Javascript. In my iteration, I used Javascript and jQuery to build a shopping list app that I wireframed.

Quiz App

A self designed quiz on video games. This quiz uses Javascript objects to contain each question text, answers and correct answer. Test your gaming knowledge and learn some interesting facts!

Video Game Character Search

Built using the API from GiantBomb.com this app lets a user search for a video game character and see a graph of the number of video games they have appeared in each year. This chart is generated using Charts.js.
